# 1001-IDEAS

A daily writing project documenting one idea per day from *1000 Ideas That Changed the Way We Think*.

## Project Structure

- `_posts/`: Daily entries in Markdown
- `assets/images/`: Uploaded image assets
- `.pages.yml`: Pages CMS content model
- `_layouts/` and `_includes/`: Jekyll layout overrides
- `.github/workflows/compress-images.yml`: Automated image compression workflow

## Publishing

This site is built with GitHub Pages (Jekyll + Minima theme) and published from this repository.
